# Student Management System 

A simple **CRUD application** for managing students.  
Built with **Java 21**, **Spring Boot MVC**, **Spring Data JPA**, **MySQL**, and **Thymeleaf Template Engine**.  
This application allows you to:
- Add new students
- View all students
- Edit existing student details
- Delete students
